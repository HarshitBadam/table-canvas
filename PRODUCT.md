# Product

## Register

product

## Platform

web

## Users

People who work with spreadsheet and tabular data across analysis, operations, and technical workflows. They need to transform, inspect, and communicate data without losing track of the logic behind it.

## Product Purpose

Table Canvas is a local-first visual data workbench for importing spreadsheet data, transforming it, exploring it, and producing reports. Success means users can create reproducible pipelines, find insights, and share results while keeping the transformation logic visible.

## Positioning

Make spreadsheet logic visible, traceable, and repeatable.

## Brand Personality

Technical, precise, and powerful. The product should give users confidence that complex data work is understandable and under control.

## Anti-references

Avoid generic, overdecorated SaaS-dashboard styling.

## Design Principles

- Make every transformation legible: the path from source data to result should be easy to follow.
- Keep powerful workflows approachable without concealing their technical substance.
- Treat reproducibility as a first-class outcome, not an implementation detail.
- Earn user trust through precise feedback, predictable behavior, and clear data state.
- Keep the interface focused on meaningful work rather than decorative dashboard conventions.

## Accessibility & Inclusion

Meet WCAG 2.2 AA, with keyboard access and reduced-motion support as baseline requirements.
